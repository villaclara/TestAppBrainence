=======
Сценарій роботи програми:

Вибираємо файл -> Натискаєм Browse -> Вводимо слово -> Натискаєм Search -> Отримуємо результат*

=======
Примітки:
* Не знаю як зробити, щоб відображення нових даних в  базі даних обновлялось при кліку на кнопку.  
Тому потрібно не обновляти сторінку, а ще раз скомпілювати програму. Тоді зміни в бд будуть показуватись.
